Cleans and formats a subnetwork in the KN into a set of flat files for distribution.

Utilities for fetching and cleaning data in the KN mySQL database and then
formatting it into a set of files that can be easily distributed.  If the output is 
not already present, queries the mysql database for a particular subnetwork, 
cleans/normalizes the data using default recommendations, reformats the data and metadata
into .edge and .json files, and finally writes these files to specificied directory (
including S3).

Arguments:
	edge_type (str): Limits edge types that are retrieved.
	species (int): The species taxon id to retrieve data for.
	edge_threshold (float): Limits edges based on weight.
	mysql host (str): mysql server credentials
	mysql_user (str): mysql server credentials
	mysql_port (int): mysql server credentials
	mysql_pass (str): mysql server credentials
	normalization/cleaning method (str): "default" using guidance by edge_type table 
	output_format (str): Specifies the output format. ".edge", ".df", etc
	output_location (str): Directory / S3 bucket to put the resulting files.
	output_credentials (str): S3 bucket access string

Output:
	Meta File: .../edge_type.species/network_metadata.json
		number_of_genes
		number_of_properties
		number_of_edges
		degree_distribution
		score_distribution
		unmapped_stats ???
		provenance:
			network_build
				commit
				command_line
				date
			subnetwork_extraction
				commit
				command_line
				date
			source_files:
				source_file_ID
				alias 
				(maybe also release version)
				remote_urls 
				checksums 
		file_checksums
			node_file
			edge_file

	Nodes File: .../edge_type.species/node_map.json
		list of all nodes
			id
			human-readable symbol
			human-readable desc
			url_reference

	Edge File: .../edge_type.species/cleaned_network.edge
		node_1_id	node_2_id	weight	edge_type	source_file_ID	source_file_line
